利用hexo和github搭建自己的免费博客

  • 其他
  • 技术

一直有搭建自己博客的想法，偶然看到了一个博客觉得页面风格十分好看，而且可以利用Github Pages来搭建个人博客，免去申请个人主机费用，于是就决定搭建个人博客了。
本文假设你是一只程序猿，知道什么是Git,NodeJs

1.用到的原料

GitHub账号
NodeJs
Hexo
一个属于自己的域名

2.安装hexo到本地

假设我们的目录在F盘，首先我们通过npm安装hexo,然后新建我们的第一个博客

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server -p 9000

完成以上动作之后，用浏览器打开localhost:9000就能看到下面的画面

3.安装yelee主题

第一次接触hexo就是因为被yelee主题吸引了，他的开源地址是https://github.com/MOxFIVE/hexo-theme-yelee
首先把项目clone到日志根目录下的themes文件夹
git clone https://github.com/MOxFIVE/hexo-theme-yelee.git themes/yelee
然后在博客根目录下修改配置文件_config.yml，使hexo使用yelee主题
theme: yelee
重新生成静态页面并启动服务器
hexo g
hexo server -p 9000
访问http://localhost:9000,能看到新的主题被应用了

4.开通Github Pages

Github Pages是github提供的免费博客服务，你可以把静态页面放到上面
首先你得申请个Github的账户(申请过程略),然后新建一个repository

repository的名称需要按照用户名.github.io格式来命名(请忽略下图红色的提示文字，因为是演示，实际的页面已经存在了)

最后打开浏览器，访问http://你的用户名.github.io 就能看到自己的Github Pages

5.绑定自己的域名

首先去网络服务供应商申请一个属于自己的域名，这里我用了腾讯云的域名服务 https://www.qcloud.com/product/dm.html
选择了.xyz的域名，一方面看上去帅，一方面是不用备案。
然后进入域名管理，点击解析去配置自己的域名，这里要注意的是腾讯需要你做实名认证才能进入解析页面。

接下来添加两条解析记录，等待十来二十分钟后生效

6.在博客项目添加CNAME文件

在项目的source文件夹下添加一个CNAME文件，把你申请的域名添加到文件里面

7.发布你的项目到Github

要把项目发布到github上，首先需要配置博客根目录下的_config.yml文件,修改deploy项

repo 配置为你的项目地址
最后在博客根目录下执行以下命令，把博客发布到Github上去
hexo clean
hexo generate
hexo deploy

如果顺利的话，这时候就能通过域名直接访问到你的主页了~！

备注：这篇文章算是后记，上述的过程有可能存在纰漏，但是思路是没错的，如果建立博客的过程中发现问题，大家请多点百度~
